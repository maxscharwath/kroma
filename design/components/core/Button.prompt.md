Primary action button for LUMA — use for play, request, confirm. One `primary` per view; secondary actions use `glass` or `ghost`.

\`\`\`jsx
<Button variant="primary" size="md" icon={<PlayIcon/>}>Lecture</Button>
<Button variant="glass">Plus d'infos</Button>
\`\`\`
Variants: primary | glass | ghost. Sizes: sm | md | lg.
